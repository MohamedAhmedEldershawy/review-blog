# 📝 Neubrutalism Review Blog

A responsive review blog built with HTML5 and CSS, featuring a Neubrutalism-inspired design.

## 🎥 Demo

https://github.com/user-attachments/assets/ae2cd5a6-fa66-4207-88a3-cf7c5add5591



## 🚀 Features

* Responsive layout for mobile, tablet, and desktop screens.
* Neubrutalism-inspired UI with bold borders and box shadows.
* CSS hover effects.
* Layout structuring using CSS positioning.
* Clean and semantic HTML structure.

## 🛠️ Technologies Used

* HTML5
* CSS
* Responsive Design (Media Queries)
* CSS Positioning
* Hover Effects

## 🧠 Challenges & Learnings

The main challenge in this project was implementing the Neubrutalism style while keeping the layout responsive and visually consistent across different screen sizes.
